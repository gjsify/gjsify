gjsify Static File Server

This page is served by a static file server built with Node.js APIs, running on both Node.js and GJS.

Features

  • File serving via fs.createReadStream().pipe(res)
  • MIME type detection via path.extname()
  • Gzip compression via zlib.gzipSync()
  • Directory listing
  • Security: directory traversal prevention
  • Conditional requests (If-Modified-Since / 304)

Test Files

  • style.css — Stylesheet (text/css, gzip-eligible)
  • data.json — JSON data (application/json, gzip-eligible)
  • hello.txt — Plain text file
